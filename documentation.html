About Frimouss Admin template

Frimouss is a fully responsive admin html template, perfect for anyone looking to start a backend system with a modern interface. No more separate websites for mobile, as this theme will adapt seamlessly to any device screen and on top of that...

Pages:

  1. Main Dashboard
  2. Search
  3. Mailbox
  4. Support
  5. Invoice
  6. Login Page
  7. User Profile
  8. Error 404


Changelog:

  • v1.0 (04.02.2013)
    • Initial release

Features:

  1. Based on Twitter Bootstrap
  2. Responsive Design, looks great on: Desktops, Tablet Devices, Mobile Devices
  3. Responsive two column layout (hideable sidebar)
  4. Fixed top bar
  5. HTML5/CSS3
  6. Custom search page
  7. Custom mailbox
  8. Charts (lines, bars, pies, Cercular, Sparklines)
  9. Many Widgets, ex. Weekly Stats, Last Users, To-Do List, Support Tickets, Chats
  10. Chat/Support page
  11. User page (static)
  12. Drag&Drop Multiupload
  13. Datepicker, Timepicker, Date rangePicker
  14. Image grid
  15. 4 icon sets (Fontello, glyphicons & splashy icons)
  16. Custom 404 error page
  17. Integrated calendar (jQuery FullCalendar)
  18. Tables (static, dynamic)
  19. Notifications (sticky, modal boxes)
  20. Responsive file explorer
  21. WYSIWYG editor with integrated file explorer
  22. Regular and extended form elements (sliders, progresbars, enchanced select elements, 2 column multiselect, styled form elements, colorpicker, masked inputs, tags handler and many more)
  23. Tooltips (always visible in the viewport), Popovers
  24. Bootstrap Javascript Plugins (modal, dropdown, tab, popover, collapse, typeahead, rowlink, fileupload )
  25. Form Validation
  26. Step by Step Wizard
  27. Drag&Drop Widget Boxes

External links

There is a js function (frimouss_external_links.init() located in frimouss_common.js) that changes all external links (icon and attribute target="_blank" is added). To disable this function for single link add ext_disabled class to this link: <a href="http://url.com" class="ext_disabled">.

Sidebar

Frimouss Admin has two columns by default, main content and a hideable sidebar. This sidebar can be hidden on page load by adding sidebar_hidden class to body.
In this template sidebar state is saved to cookie. If you don't want this functionality just coment/remove some code in js/frimouss_common.js.
There is also implemented scrollbar in this sidebar, if your content in sidebar is longer than window height then this scrollbar will show up.
If you want to move sidebar to right add sidebar_right to body: <body class="sidebar_right">

Layout, grid & elements

This template has 25 HTML pages, lot of jQuery plugins and custom widgets, elements from twitter bootstrap, form elements and much more… This template is using jQuery cookie to remember when you switch sidebar position, change navigation, switch fluid/fixed layout. So when you restart browser or refresh/change page it will remember latest choice. It can also be done manually by adding single classes that is explained in documentation. Most of elements was designed directly in browser, while psd file contain only main elements.
